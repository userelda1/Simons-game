A simple Simon Game built using HTML, CSS, JavaScript, and jQuery.
The game generates a random sequence of colours, and the player must repeat the pattern correctly to progress to the next level.

Technologies Used
HTML
CSS
JavaScript 
jQuery


How to Play
Press any key to start the game.
Click the buttons in the same order the game shows.
Each level adds one new colour to the pattern.
If you click the wrong colour, the game ends.
Press a key again to restart.
